# Literal & Keyword

## Literal

<div className="table-wrapper">
| 種類 | パターン | 例 |
| --- | --- | --- |
| 数値 | `-?(\d+\.?\d*\|\.\d+)((i\|u)(8\|16\|32\|64\|128)\|f(32\|64))?` | `12`, `-35i64`, `.2f32` |
| 文字列 | `".*"` | `"Hello, world!"` |
| シンボル | `'.+` | `'foo`, `'シンボル` |
| 引数 | `#\d+` | `#12` |
| ディレクティブ | `/.+` | `/exit` |
</div>

## Keyword

| 表記 | 意味 |
| --- | --- |
| `T` | 真値 |
| `##` | 自己参照 |

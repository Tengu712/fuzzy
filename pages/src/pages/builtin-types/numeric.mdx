# numeric

## General

数値型は以下の3分類全12種類を含む。

| 分類 | 該当型 |
| --- | --- |
| 符号付き整数 | `i8`, `i16`, `i32`, `i64`, `i128` |
| 符号なし整数 | `u8`, `u16`, `u32`, `u64`, `u128` |
| 浮動小数点数 | `f32`, `f64` |

## Functions

### `->`: `@['symbol]`

可変変数を定義する。
返戻値は`()`。

```fuzzy
42 -> 'num. -- numという可変変数が定義される
num         -- 42
```

### `=>`: `@['symbol]`

不変変数を定義する。
返戻値は`()`。

```fuzzy
3.14f32 => 'pi. -- piという不変変数が定義される
pi              -- 3.14
```

### `:`: `@['symbol]`

型キャストを行う。

```fuzzy
42 : 'f64      -- 42.0
3.14f32 : 'i32 -- 3
```

### `+`: `@[SAME-AS-SUBJECT]`

主語と目的語の和を返す。
ただし、目的語の型は主語と同じである。

```fuzzy
1 + 2         -- 3
3.f32 + .5f32 -- 3.5
1 + 2.0f32    -- エラー
```

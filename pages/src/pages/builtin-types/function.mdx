# function

## General

関数型は引数の型を表す`'symbol`値あるいは関数型を羅列した配列によって表記する。
REPLや本ドキュメントでは`@[引数1 引数2 ...]`のように表示される。

## Functions

### `->`: `@['symbol]`

可変変数あるいは可変関数を定義する。
目的語が`'型名:関数名`の形式を取る場合、`'型名`に`関数名`を定義する。
返戻値は`()`。

```fuzzy
{ 1 + 2 } : [], -> 'f.   -- fという可変変数が定義される
{ 1 + 2 } : [], -> 't:f. -- tの上にfという可変関数が定義される
```

### `=>`: `@['symbol]`

不変変数あるいは不変関数を定義する。
目的語が`'型名:関数名`の形式を取る場合、`'型名`に`関数名`を定義する。
返戻値は`()`。

```fuzzy
{ 1 + 2 } : [], => 'f.   -- fという不変変数が定義される
{ 1 + 2 } : [], => 't:f. -- tの上にfという不変関数が定義される
```

### `@`: `@[ARGUMENT1 ARGUMENT2 ...]`

目的語を引数に取り関数を評価する。

```fuzzy
{ 1 + 2 } : [], -> 'f.
f @                               -- 3
{ #0 + #1 } : ['i32 'i32], -> 'f.
f @ 1 2                           -- 3
```

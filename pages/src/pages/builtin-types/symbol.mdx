# symbol

## General

シンボル型。
`'`で始まる識別子。

## Functions

### `!`: `@[]`

改行なしで標準出力する。
返戻値は主語。

```fuzzy
'foo ! -- fooと出力される
```

### `!!`: `@[]`

改行ありで標準出力する。
返戻値は主語。

```fuzzy
'foo !! -- foo
        -- と出力される
```

### `->`: `@['symbol]`

可変変数を定義する。
返戻値は`()`。

```fuzzy
'hello -> 'sym. -- symという可変変数が定義される
sym             -- 'hello
```

### `=>`: `@['symbol]`

不変変数を定義する。
返戻値は`()`。

```fuzzy
'world => 'name. -- nameという不変変数が定義される
name             -- 'world
```

### `%`: `@[]`

シンボルを評価する。

```fuzzy
12 -> 'a. -- aという名前の変数を定義する
'a %      -- aとみなされる
          -- 12
```

### `<`: `@['symbol]`

主語が目的語より小さいかを判定する。

```fuzzy
'a < 'ab -- T
'2 < '10 -- ()
```

### `<=`: `@['symbol]`

主語が目的語以下かを判定する。

```fuzzy
'a <= 'ab -- T
'2 <= '10 -- ()
```

### `>`: `@['symbol]`

主語が目的語より大きいかを判定する。

```fuzzy
'a > 'ab -- ()
'2 > '10 -- T
```

### `>=`: `@['symbol]`

主語が目的語以上かを判定する。

```fuzzy
'a >= 'ab -- ()
'2 >= '10 -- T
```

### `==`: `@['symbol]`

主語が目的語と同値かを判定する。

```fuzzy
'a == 'a  -- T
'a == 'ab -- ()
```

### `!=`: `@['symbol]`

主語が目的語と異値かを判定する。

```fuzzy
'a != 'a  -- ()
'a != 'ab -- T
```

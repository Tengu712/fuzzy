# bool

## General

真理値を表す型。次の2種類の値のみを持つ。

| 表記 | 意味 |
| --- | --- |
| `T` | 真 |
| `()` | 偽 |

## Functions

### `->`: `@['symbol]`

可変変数を定義する。
返戻値は`()`。

```fuzzy
T -> 'a. -- aという可変変数が定義される
a        -- T
```

### `=>`: `@['symbol]`

不変変数を定義する。
返戻値は`()`。

```fuzzy
T => 'b. -- bという不変変数が定義される
b        -- T
```

### `~`: `@[]`

否定を返す。

```fuzzy
T ~  -- ()
() ~ -- T
```

### `&&`: `@['bool]`

主語と目的語の論理積を返す。

```fuzzy
T && T   -- T
T && ()  -- ()
() && T  -- ()
() && () -- ()
```

### `||`: `@['bool]`

主語と目的語の論理和を返す。

```fuzzy
T || T   -- T
T || ()  -- T
() || T  -- T
() || () -- ()
```

### `>>`: `@['{}]`

主語が`T`であるとき、目的語の遅延ブロックを評価する。
返戻値は主語。

```fuzzy
1 == 1 >> { "true" !! } -- trueと出力される
                        -- T
0 == 1 >> { "true" !! } -- 何も出力されない
                        -- ()
```

### `!>`: `@['{}]`

主語が`()`であるとき、目的語の遅延ブロックを評価する。
返戻値は主語。

```fuzzy
1 == 1 !> { "false" !! } -- 何も表示されない
                         -- T
0 == 1 !> { "false" !! } -- falseと表示される
                         -- ()
```


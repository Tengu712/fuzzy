# Fuzzy, a joke programming language.

Fuzzyは一種のジョークプログラミング言語です。

- 中置記法
- 記号のみからなるビルトイン関数名
- やや強い型付けによって一意に定まるわかりづらい文法

```fuzzy
-- FizzBuzz. --
1 -> 'i.
{ i <= 100 } %% {
    i % 3, == 0 >> { "Fizz" ! }
    i % 5, == 0 >> { "Buzz" ! }
    i % 3, == 0 || i % 5, == 0; >> { "" !! } !> { i !! }
    i + 1, -> 'i.
}
```
